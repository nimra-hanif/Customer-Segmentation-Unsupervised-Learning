Task Objective: The objective of this project is to perform customer segmentation using an unsupervised learning approach. The aim is to group customers based on their annual income and spending behavior in order to identify different customer segments and support targeted marketing strategies.

Approach: The project begins with data preprocessing, where missing values and duplicates are handled, and relevant features such as annual income and spending score are selected. The data is then scaled to ensure all features are on the same range. After preprocessing, exploratory data analysis (EDA) is performed using histograms, scatter plots, and box plots to understand data distribution, relationships, and outliers. 
Next, the K-Means clustering algorithm is applied to segment customers into different groups. The Elbow Method is used to determine the optimal number of clusters. Once the model is built, customers are assigned to clusters based on their similarity. For better understanding and validation, cluster visualizations are created using scatter plots as well as dimensionality reduction techniques like PCA or t-SNE.

Result and Findings: The analysis successfully identifies distinct customer segments based on income and spending patterns. These include high-income high-spending customers who are valuable for the business, high-income low-spending customers who have potential for increased engagement, low-income high-spending customers who may respond well to discounts, and low-income low-spending customers who require awareness strategies. A moderate segment of average customers is also observed.
These findings help businesses understand customer behavior more effectively and enable them to design personalized marketing strategies. Overall, this project demonstrates how machine learning techniques can be used to generate meaningful insights and support data-driven decision-making for business growth.
